# Task 2 – Student Digital Portfolio

## Platforms Set Up

### 1. GitHub (github.com)
GitHub is a version control and code hosting platform used by developers worldwide to store, share, and collaborate on code projects. I created a profile README repository (a special public repository named after my username) that displays on my GitHub profile page.

**Profile README includes:**
- My name and registration number
- Branch and year
- What I hope to learn during my B.Tech

**Screenshot:** `github-profile.png`

### 2. LinkedIn (linkedin.com)
LinkedIn is the world's largest professional networking platform. It is used to connect with employers, find internships, and build a professional identity. I filled in the Education section with my current degree (B.Tech), institution (VIT Bhopal University), and expected graduation year.

**Screenshot:** `linkedin-profile.png`

---

### 3. Kaggle (kaggle.com)
Kaggle is an online community for data scientists and machine learning practitioners. It hosts datasets, competitions, and learning notebooks. Even as a first-year student, having a Kaggle profile shows interest in data science and AI — highly relevant fields for CSE students.
